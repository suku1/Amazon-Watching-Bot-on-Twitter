# Amazon-Watching-Bot-on-Twitter
OculusRiftS入荷情報bot
https://twitter.com/oculusrifts_bot  
の中身です.  
試してはいませんが, URLとか価格とか書き換えると他の商品にも使えると思います. 

## Requires  
**使用モジュール**  
 - Requests
```bash
$ pip install requests
```
 - BeautifulSoup4
```bash
$ pip install bs4
```

## Usage  
APIkey.jsonにTwitterAPIのkeyを設定すれば動きます.  
```bash
$ python bot.py
```
